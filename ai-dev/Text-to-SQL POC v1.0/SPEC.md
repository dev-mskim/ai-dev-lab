# 프로그램 사양서 (Specification)

> 프로젝트명: Text-to-SQL ERP 챗봇
> 작성일: 2026-03-26
> 개발자: 1인 개발 (자기계발 → ERP 확장 목표)

---

## 1. 프로젝트 목적

| 구분 | 내용 |
|------|------|
| 단기 목표 | 자연어로 질문하면 DB 결과를 반환하는 데모 프로그램 제작 |
| 장기 목표 | 현재 사용 중인 Oracle ERP에 AI 챗봇 기능 추가 |

---

## 2. 핵심 기능 (무엇을 만드나)

```
사용자가 자연어로 질문
        ↓
AI가 SQL을 자동 생성
        ↓
DB에서 실행 후 결과 반환
        ↓
결과를 자연어로 요약해서 답변
```

### 기능 목록

| # | 기능 | 설명 | 상태 |
|---|------|------|------|
| F01 | 자연어 입력 | 채팅창에 한국어로 질문 입력 | ⬜ 개발 예정 |
| F02 | SQL 자동 생성 | 질문을 분석해 SQL 자동 생성 | ⬜ 개발 예정 |
| F03 | SQL 실행 | 생성된 SQL을 DB에서 실행 | ⬜ 개발 예정 |
| F04 | 결과 요약 | DB 결과를 자연어로 요약 | ⬜ 개발 예정 |
| F05 | 스키마 검색 | 질문과 관련된 테이블 자동 탐색 | ✅ 완료 |
| F06 | SQL 안전 검증 | SELECT 외 쿼리 차단 | ⬜ 개발 예정 |
| F07 | 대화 히스토리 | 이전 대화 내용 화면에 유지 | ⬜ 개발 예정 |
| F08 | SQL 표시 | 생성된 SQL을 화면에 함께 표시 | ⬜ 개발 예정 |

## 3. 기술 스택

| 역할 | 기술 | 버전 | 비고 |
|------|------|------|------|
| 언어 | Python | 3.10 | |
| AI | Claude API (anthropic) | 0.86.0 | Sonnet 4.6 모델 사용 |
| 웹 UI | Streamlit | 1.55.0 | 별도 HTML 불필요 |
| 데이터베이스 (데모) | SQLite | 내장 | 파일 기반, 설치 불필요 |
| 데이터베이스 (실제) | Oracle | - | ERP 연동 시 전환 |
| DB 연결 | SQLAlchemy | 2.0.48 | Python용 JPA/MyBatis |
| Vector DB | ChromaDB | 1.5.5 | 스키마 검색용 |
| 임베딩 모델 | sentence-transformers | 5.3.0 | 다국어 지원 |
| Oracle 드라이버 | cx-Oracle | 8.3.0 | ERP 연동 시 사용 |
| 환경변수 관리 | python-dotenv | 1.2.2 | application.properties 역할 |

## 4. 데이터 구성

### 데모용 샘플 데이터 (SQLite)

| 테이블 | 설명 | 건수 |
|--------|------|------|
| CUSTOMERS | 고객 정보 | 8건 |
| PRODUCTS | 상품 정보 | 8건 |
| ORDERS | 주문 정보 | 12건 |
| ORDER_ITEMS | 주문 상세 | 16건 |

### 질문 예시

```
"이번 달 매출 TOP 5 고객 알려줘"
"서울 지역 고객 목록 보여줘"
"전자기기 카테고리 상품 중 가장 비싼 것은?"
"홍길동 고객의 전체 구매 내역"
"월별 매출 추이 보여줘"
```
